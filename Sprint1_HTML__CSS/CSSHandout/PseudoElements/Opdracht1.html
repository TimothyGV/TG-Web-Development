Opdracht 1

voorbeelden van Pseudo Elements

::before plaatst content voor de content van een element

::after plaatst content achter de content van een element

::marker target de marker (het icoontje voor de li) van een ul/ol li.

::selection geeft styling aan het stuk dat geselecteert wordt door de gebruiker

Verschil tussen pseudo-elementen en pseudo-class. Elements worden gebruikt om een specifiek element of delen daarvan te stylen. Classes zijn een selector die helpen in de selectie van iets, onder een bepaalde conditie. Bijvoorbeeld link::visited. Daardoor bestaan zij niet echt in de 'document tree'.
