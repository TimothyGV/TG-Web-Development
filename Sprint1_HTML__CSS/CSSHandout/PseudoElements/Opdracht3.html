Met de ::target-text pseudo-element wordt een text segment gestyled waar naar gescrolled wordt, indien gebruik gemaakt wordt van scroll-to-text. Niet elke browser ondersteunt dit.

Voorbeeld

p ::target-text { background-color: rebeccapurple; color: white; font-weight: bold; }
