Transitions

Zoek uit wat Transitions zijn en beschrijf met eigen woorden

CSS Transitions zijn een visueel effect waarmee eigenschappen van css elementen kunnen worden aangepast, vergelijkbaar met pseudo-elementen :hover of :focus. In dit geval kan er echter een tijdsvariabele aan toegevoegd worden, die er voor zorgt dat het effect over een bepaalde periode wordt uitgevoerd. Kort: in plaats van direct effect (hover), duurt het effect een aantal seconden. Dit zorgt voor een visueel prettiger geheel. Dit wordt vaak toegepast op background-color, width, height en opacity.

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
