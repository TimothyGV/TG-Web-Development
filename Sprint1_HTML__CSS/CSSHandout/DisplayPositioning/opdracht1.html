Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

Er zijn vijf verschillende position property's: static relative fixed absolute sticky

Static

Static elementen worden niet geraakt door eigenschappen van top, bottom, left en right. Het heeft altijd zijn vaste plaats in de flow van de pagina.

Relative

Met relative wordt het element geplaatst ten opzichte van zijn normale positie. Je kan hem dus verplaatsen met top, bottom, left en right.

Fixed

Met fixed wordt het element vastgezet relatief tenopzichte van de viewport. Het blijft dus altijd op dezelfde plek staan, zelfs wanneer de pagina scrolled.

Absolute

Met absolute wordt het element relatief ten opzichte van zijn parent geplaatst. kan verplaatst worden binnen de parent, en verschuift mee met de parent. Indien de parent ontbreekt gebruikt het de body. Absolute elementen kunnen overlap hebben met andere elementen

Sticky

Een sticky element varieert tussen fixed en relative. De positie is dan afhankelijk van de scrolling positie van de gebruiker.

Default waarde

De default is altijd static
