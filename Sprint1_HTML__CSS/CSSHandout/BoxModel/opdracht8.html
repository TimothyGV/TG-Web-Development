Dit is de content met box-sizing

Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

Box-sizing: border box

In de voorgaande opdracht hebben we gezien dat padding, margin en border bij elkaar optellen. Hierdoor wordt de box dus als het ware groter. Door border box gebeurt dat niet. Padding en border worden in de box toegevoegd en de width en height geldt voor alle property's..
