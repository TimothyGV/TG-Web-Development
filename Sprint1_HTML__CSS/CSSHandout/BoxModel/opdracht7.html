1. Hoe breed is de box?

Totaal element breedte = width (100) + left padding (5) + right padding (5) + left border (5) + right border(5) + left margin(5) + right margin(5 )=130px

2. Hoe hoog is de box?

Totaal element hoogte = height (100) + top padding(5) + bottom padding(5) + top border(5) + bottom border(5) + top margin(5) + bottom margin(5)

3. Wat merk je op met width en height?

Wanneer je de width en height gebruikt, worden alleen de breedte en hoogte van de content area aangepast. Voor de volledige grootte van het element, moet je ook nog de padding, borders en margins toevoegen.
