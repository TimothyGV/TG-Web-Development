Display property

Wat gebeurt er als je een width en height toevoegt aan de span?

Inline-block elementen bestaan in dit geval binnen het 'block'. Width en height kunnen, in tegenstelling tot, inline wel toegepast worden. Het verschil met display:block is dat er geen nieuwe regel wordt gemaakt, en de inline-block dus naast andere elementen staat. Het parent element met Display:block, wordt dus naar buiten 'geduwd'

Dit is een block box en neemt de hele breedte in waardoor de volgende box op de volgende regel zit.
