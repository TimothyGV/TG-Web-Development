Als je ruimte tussen de border en buiten de box wilt toepassen, gebruik je de property margin.

Deze box heeft een padding en een border, maar geen margin.
Deze box heeft dezelfde padding en border als de vorige, maar nu ook een margin van 70px (Boven, rechts, onder, links).
Door de padding van de vorige container, wordt deze dus op een afstand van 70x naar rechts opgeschoven.
