Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

External

Antwoord Er zijn drie manier om dit te doen: 1. External, daarbij wordt verwezen naar een extern css bestand met styling. Deze wordt met onderstaande link in de head aangeroepen

link rel="stylesheet" href="mystyle.css"

Internal

Styling kan ook worden toegepast met internal commands. Deze worden in de head onder style geplaatst. Dit is handig voor korte opdrachten

Dit is bijvoorbeeld internal gestyled.

Inline

Met inline css wordt de styling in het document toegepast, bij de specifieke tag.
