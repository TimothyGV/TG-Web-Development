Cascade

Cascading is een belangrijk principe binnen css. Het is zelfs in de naam verwerkt: cascading style sheets. Het gaat om stijlbladen, die de mogelijkheid bieden om vormgeving van webpagina's in een apart document toe te passen. 'cascading' verwijst daarbij naar de styling rules, die 'cascade' (waterval) van verschillende bronnen. Kort gezegd: er zit een bepaalde hierarchie in de styling, waarbij een hogere voorganger de regels van een lagere overschrijft.

Voorbeeld cascading, met conflict:

h1 { color: blue; } h1 { color: yellow; } <h1>This is a header</h1>

Voorbeeld cascading

Hoewel de h1 een kleur van blauw krijgt, wordt deze overschreven door een hoger gelegen 'rule' van de div

.main-header { color: purple; } h1 { color: blue; } <h1 class="main-header">Dit is een header</h1>

Inherited

Inherited betekent doorgegeven. In CSS betekent dit dat een element zonder styling, de rules kan overnemen van zijn ouder.

Voorbeeld Inherited

p {color: purple;}

<p>Deze tekst heeft <span>een span</span> gekregen, maar de span heeft geen styling rule. Daarom neemt hij deze over van zijn parent 'p'</p>
