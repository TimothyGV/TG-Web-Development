Maak nu een html bestand aan met voorbeelden van alle soorten selectors erin. Maak een link in je antwoord die naar dat html bestand verwijst.

selector voorbeeld beschrijving
.div .voorbeeld Alle elementen met de klasse naam class='intro'
#id #first__section Alle elementen met de id naam id='first__section'
* selecteert alle elements * aan het begin van het css bestand selecteert alles
:root :root selecteert het root element
element p Selecteert alle <p> elementen
.class1 .class2 .voorbeeld1 .voorbeeld2 Selecteert alle elementen met voorbeeld2 die een kind zijn van een element voorbeeld1
element.class p.voorbeeld Selecteert alle <p> elementen in intro
element,element p,.voorbeeld Selecteert alle <p> en alle elementen in voorbeeld
element>element .voorbeeld > p Selecteert alle <p> waar voorbeeld een parent is
element+element .voorbeeld + p Selecteert alle <p> die direct na .voorbeeld zijn geplaatst
element~element p~ul Selecteert alle <ul> die vooraf zijn gegeaan door een <p>
:hover a:hover Selecteert en geeft effect op een link
:nth-child(n) p:nth-child(2) Selecteert elk element <p> dat het tweede kind is van element
:focus input:focus Selecteert elke input element dat de focus heeft
