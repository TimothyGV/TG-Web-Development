Extend and Inheritance

Met mixins kun je styling hergebruiken in een project een verschaft de mogelijkheid om argumenten sterker en flexibler te maken. De @extend stelt ons in staat om ook stylen te hergebruiken, maar een selector inherit die van een ander.

@extend is handig voor items die grofweg dezelfde styling hebben, maar op kleine details afwijken.

By using the @extend directive, you do not need to specify several classes for an element in your HTML code, like this: <button class="button-basic button-report">Report this</button> . You just need to specify .button-report to get both sets of styles. The @extend directive helps keep your Sass code very DRY
